# Pregunta 3

## Estructura nodo

#### En ambos casos la estructura nodo se compone de:
 * 1 int
 * 1 string
 * 1 char
 * 1 double

### Composicion del segmento a

``` 
struct nodo {
    int a;
    char b[16];
    char c;
    double d;
};

```
### Composicion del segmento b

``` 
struct nodo {
    char a;
    int b;
    char c[16];
    double d;
};

```
